## Notes For Website Use

- Hero image/video must keep upper center or upper left dark enough for headline.
- Avoid bright gold dominance. Use warm amber only as an accent.
- Tarot cards should be custom oracle cards, not classic tarot copies.
- Use `webp` for still images.
- Use 16:9 for hero backgrounds, 5:8 for cards, 1:1 or 4:5 for detail images.
- For transparent motion overlays, use WebM VP9 alpha if supported, with MP4 fallback.
